# 包包滴消消乐（修正卡頓 & 邏輯漏洞）

你回報的問題，本版已處理：
1) 圖片載入慢：加了 `<link rel="preload">` + JS 預載，減少白塊/卡頓。
2) 消除音效：每一次消除都跟滑動用同一個音效（swap.mp3）。
3) 開局/未操作就三連但不消：開局增加「自動穩定棋盤」保險機制，確保沒有殘留三連。

## 音效
- swap.mp3：滑動 + 消除（同一個）
- win.mp3：通關
（pop.mp3 不需要；我沒有放 pop.mp3 檔案）

## 建議加速（很重要）
